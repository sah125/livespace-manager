# 🏢 LiveSpace Manager

**LiveSpace Manager** is a modern property and accommodation management system designed to manage student residences, BnBs, rentals, and other living spaces in one platform.



## 🚀 Features

* 🧑‍🎓 Student Residence Management
* 🏠 BnB & Rental Management
* 🛠️ Maintenance Request System
* 📊 Weekly & Monthly Reports Dashboard
* 👨‍💼 Admin Panel
* 📩 Request Tracking & Response Time Analysis



## 🧠 Vision

To build a **scalable smart living platform** that helps property owners, institutions, and communities manage spaces efficiently using technology.


## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Bootstrap 5
* (Future) React / Node.js
* (Future) Database Integration
